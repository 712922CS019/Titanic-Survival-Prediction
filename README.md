Titanic Survival Prediction (Machine Learning)

 Overview
 
This project predicts whether a passenger survived the Titanic disaster using machine learning techniques. It uses the Titanic dataset to build classification models and analyze key factors affecting survival


 Features
 
 Data cleaning and preprocessing (handling missing values)
 Exploratory Data Analysis (EDA)
 Feature engineering (FamilySize, IsAlone, Title extraction)
 Model building using multiple algorithms
 Hyperparameter tuning for performance improvement
 Model evaluation using classification metrics


 Models Used
 
 Logistic Regression  
 Random Forest Classifier  



 Results
 
 Achieved accuracy of ~82% on test data  
 Evaluated using:
   Accuracy Score  
   Precision & Recall  
   F1 Score  
   Confusion Matrix  
   ROC-AUC Curve  



Key Insights

Gender and passenger class had a strong impact on survival  
Higher class passengers had better survival chances  
Family size influenced survival probability  



Technologies Used

Python 

 NumPy, Pandas  
 Matplotlib, Seaborn  
 Scikit-learn
